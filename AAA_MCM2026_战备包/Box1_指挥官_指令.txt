# Critical Enforcement Rules

## 1. Modeling Standards
- **ASMP_001**: If data missing → use scaling laws (e.g., Kleiber's Law Y∝M^0.75), NEVER arbitrary assumptions
- **MOD_012**: For growth/decay → derive from dX/dt = [source] - [sink]
- **EQ_009 (CRITICAL)**: If parameter varies >20% → MUST be function P(t) or P(x,y). BAN piecewise constants.
- **FBD_001**: For mechanics → draw Free Body Diagram, verify all forces

## 2. Expert Add-ons (Max 2)
- **STO_001**: If "risk/uncertainty" → Monte Carlo (N≥300) + Confidence bands
- **SPAT_001**: If "spread/distribution" → Network/PDE + Heatmap
- **OPT_002**: If conflicting objectives → Pareto Front + Knee point

## 3. Validation Protocol
- **VALID_005**: Eigenvalue analysis (Re(λ)<0 for stability)
- Phase plane plot (nullclines + vector field)
- Sensitivity test (±10% parameters)
